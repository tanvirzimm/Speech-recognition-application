## Only for laptop and desktop computer .

###### Clone this Repo :
```shell
git clone https://github.com/tanvirzimm/Speech-recognition-application.git
```
###### Install :
```shell
npm install
```

###### Run :
```shell
npm start
```
###### Build dist:
```shell
npm run build
```
